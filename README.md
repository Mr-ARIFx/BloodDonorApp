# 🩸 Blood Donor App

An Android application for managing blood donors and searching blood groups easily.

---

## 📱 Features

- User Registration
- Blood Donor Search
- Blood Group Filter
- Contact Donors
- Easy UI

---

## 🛠️ Built With

- Java
- Android Studio
- Firebase

---

## 📥 Download APK

Click below to download and install the app:

[⬇ Download APK]
https://github.com/Mr-ARIFx/BloodDonorApp/releases/download/v1.0/app-debug.apk

---

## 📲 Installation Process

1. Download the APK
2. Open the APK file
3. Allow "Install from Unknown Sources"
4. Click Install
5. Open the App
